### Week 1 Objectives:

* Connect and get acquainted with members of First Cloud Journey (FCJ).
* Understand foundational AWS concepts and core service groups (Compute, Storage, Networking, Database, ...).
* Get familiar with the AWS Management Console and basic navigation/operations via the web interface.
* Install, configure, and practice basic operations using AWS CLI.
* Start learning and practicing with Amazon EC2 (launching instances, SSH access, EBS volumes, Elastic IP).
